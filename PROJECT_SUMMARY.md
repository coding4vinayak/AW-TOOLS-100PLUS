## 🚀 How to Run

### Development
```bash
cd C:\Users\vinayak\Desktop\99tools\abetworks-tools

# Make sure NODE_ENV is not set to production
set NODE_ENV=

# Install dependencies (if needed)

# Start development server

```

### Production Build
```bash

npm run build

# Output in dist/ folder
```

The `dist` folder can be deployed to:
- Netlify

- GitHub Pages
- Cloudflare Pages
